## 1 Installation

```
$ pip install pybtm
```

## 2 Usage

### 2.1 Create entropy

get_entropy() create 128bits entropy.

```python
>>> from pybtm import key
>>> key.get_entropy()
'100e2704b431f914e3262926bdba6fce'
```
